Para este proyecto sobre mi portfolio comencé instalando python en mi escritorio luego instalé la extension de visual estudio code, para empezar a crear el directorio del proyecto le pedi a chat gpt que me diera el directorio de un proyecto de python usando flask seguí por crear un entorno virtual /venv donde instale las dependencias del archivo requirements.txt dependencias como flask, flask_mail, SQLAlchemy; luego cree el archivo .gitignore, instalé git y aprendi a utilizar el control de versiones con git creando branch y editando el codigo para luego unirlo a la raiz del proyecto con merge..luego subi me repositorio a github usando git push origin main para subir los archivos, aprendi que si hago modificaciones en una rama puedo guardar esos cambios en la rama con git add . eso hace que se compare y guarde en cache las actualizaciones y no tener que actualizar el codigo completo, luego con git commit -m “descripcion de la actualización” podemos ahora si crear una version del codigo en la rama que estemos usando eso hace que los cambios queden guarados en una versión a parte de su edición anterior y que podemos unir a la rama madre o raiz/main y luego subir el código a github con git push origin main. Dentro de git puedo desplegar mi codigo de manera publica para que cualquiera que tenga el link tenga acceso al repositorio
